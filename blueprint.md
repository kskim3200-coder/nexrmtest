# Project Blueprint

## Overview

This project is a simple lottery number generator. It generates 6 unique random numbers between 1 and 45.

## Current State

### Style, Design, and Features

*   **Initial Version:**
    *   Basic HTML structure with a title, a container for the numbers, and a "Generate" button.
    *   Styling for the container, numbers, and button to be visually appealing.
    *   JavaScript logic to generate and display the lottery numbers.

## Current Request

### Plan

1.  **`index.html`**: Create the basic HTML structure for the lottery generator.
2.  **`style.css`**: Add CSS to style the page, the number display, and the button.
3.  **`main.js`**: Implement the JavaScript logic to generate 6 unique random numbers from 1 to 45 and display them on the page.
